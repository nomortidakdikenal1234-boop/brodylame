Memeriksa status database...

Cara Mengaktifkan di Firebase Console Anda:

1. Aktifkan Anonymous Auth (Sangat Disarankan)

  1. Buka Firebase Console proyek shin-taeyoung-website Anda.
  2. Pilih menu Authentication -> tab Sign-in method.
  3. Klik Add new provider, pilih Anonymous, lalu aktifkan (Enable) dan simpan.

2. Aturan Membaca/Menulis Realtime Database

  1. Pilih menu Realtime Database -> tab Rules.
  2. Pastikan aturan membaca dan menulis disetel agar mengizinkan akses. Contoh aturan default: 
    {
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
